# 🚀 My Portfolio Website

A modern portfolio website built using React, TypeScript, and Vite.

## 🔥 Features
- Responsive design
- Projects showcase
- Experience & achievements section
- Clean UI with Tailwind CSS

## 🛠️ Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
